Каждая таблица соответствует **1-й нормальной форме**, так как все атрибуты атомарные, т.е. каждый кортеж содержит только одно значение для каждого атрибута.  
В дальнейшем, для каждой таблицы соответствие (n+1)-й нормальной форме подразумевает ранее доказанное соответствие n-й нормальной форме и требование к этого соответствия будет опускаться.
